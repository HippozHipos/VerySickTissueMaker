\doxysubsection{Detailed Description}
System that Renders the all the meshes. 

This class iterates over all the \doxylink{classbe_1_1_mesh_renderer}{Mesh\+Renderer} s in the scene and renders them.

\doxylink{classbe_1_1_mesh_renderer}{Mesh\+Renderer} can be added to the scene by calling \doxylink{classbe_1_1_scene_container_ac90d71a74d89b19906903f5462de30b9}{Scene\+Container.\+Add$<$\+Mesh\+Renderer$>$()} or by calling Add on any classes that inherit from \doxylink{classbe_1_1_scene_container}{Scene\+Container}.

How it works\+: When rendering, it iterates through every \doxylink{classbe_1_1_camera}{Camera} in the scene. If the camera is active, it renders to the \doxylink{classbe_1_1_camera_a352364ff86f498f2bde5d507dc93aabd}{Camera\+::target\+\_\+viewport}. Currently, if the \doxylink{classbe_1_1_camera_a352364ff86f498f2bde5d507dc93aabd}{Camera\+::target\+\_\+viewport} is "{}\+Main Platform Window"{}, it simply unbinds the frame buffers so that rendering is done to the default frame buffer. This will change in the future. Otherwise, it renders to the frame buffer of specified window. The texture of the specified frame buffer is then passed to imgui for the window that it belongs to by \doxylink{classbe_1_1_b_e_im_gui_viewport_a848f636565597040649e70fe5d3da69c}{BEIm\+Gui\+Viewport\+::\+Render}.
