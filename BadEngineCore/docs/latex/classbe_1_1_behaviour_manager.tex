\doxysubsection{Detailed Description}
Manages behaviours of scene objects by calling specific behaviour methods. 

See related\+: \doxylink{classbe_1_1_behavioural_scene_object}{Behavioural\+Scene\+Object}

The \doxylink{classbe_1_1_behaviour_manager}{Behaviour\+Manager} class manages the behaviour method invocations of objects that inherit from \doxylink{classbe_1_1_behavioural_scene_object}{Behavioural\+Scene\+Object} class. It allows adding, removing, and invoking various methods related to object behaviour, such as Pre\+Start, Start, Update, UI, End, and Post\+End.

The behaviour methods are checked using SFINAE (e.\+g., {\ttfamily \doxylink{structbe_1_1_has_pre_start_method}{Has\+Pre\+Start\+Method}}) and are only invoked if they are present in the object.

The following behaviour methods can be managed\+:
\begin{DoxyEnumerate}
\item Pre\+Start -\/ Called once when the object is created.
\item Start -\/ Called once after Pre\+Start.
\item Update -\/ Called every frame.
\item UI -\/ Called every frame for UI rendering (Im\+Gui).
\item End -\/ Called once when the object is deleted.
\item Post\+End -\/ Called once after End. 
\end{DoxyEnumerate}
