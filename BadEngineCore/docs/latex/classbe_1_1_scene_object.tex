\doxysubsection{Detailed Description}
Represents an object in the scene. 

This class can be inherited from by any class that will be in the scene without any behaviour. It inherits from \doxylink{classbe_1_1_scene_container}{Scene\+Container} and provides access to the objects stored by \doxylink{structbe_1_1_application_container}{Application\+Container} on top of it.

Each \doxylink{classbe_1_1_scene_object}{Scene\+Object} also has a \doxylink{structbe_1_1_transform}{Transform} object added to it by default.
