\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Derived$>$\newline
class be\+::\+Behavioural\+Scene\+Object$<$ Derived $>$}
Base class for objects in the scene with behaviour. 

See Related \doxylink{classbe_1_1_behaviour_manager}{Behaviour\+Manager}

This class can be inherited from by any class in the scene that has behaviour. Any classes that inherit from this class will be able to define the following methods that will be automatically called.


\begin{DoxyEnumerate}
\item Pre\+Start -\/ Called once when the object is created
\item Start -\/ Called once after Pre\+Start
\item Update -\/ Called once every frame
\item UI -\/ Called once every frame. Any Im\+Gui calls need to be made in this function.
\item End -\/ Called once when the object\textquotesingle{}s scope ends (i.\+e. when it\textquotesingle{}s deleted)
\item Post\+End -\/ Called once after End. The object is destroyed after this.
\end{DoxyEnumerate}
