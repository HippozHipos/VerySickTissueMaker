\doxysubsection{Detailed Description}
Represents a container that holds any object that might be in the scene. 

See related\+: \doxylink{classbe_1_1_scene_object}{Scene\+Object}

Any object that is added to this container becomes one of its children and this container becomes that objects parent. NOTE\+: Any object that can be added to this container should also inherit from \doxylink{classbe_1_1_scene_container}{Scene\+Container}. Therefore, you can build hierarchical tree structure of objects with it. Example\+:
