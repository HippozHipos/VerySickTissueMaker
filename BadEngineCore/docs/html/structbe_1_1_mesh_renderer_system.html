Detailed Description

System that Renders the all the meshes.

This class iterates over all the MeshRenderer s in the scene and renders them.

MeshRenderer can be added to the scene by calling SceneContainer.Add<MeshRenderer>() or by calling Add on any classes that inherit from SceneContainer.

Example

class Player : public SceneObject
{
public:
Player()
{
MeshRenderer& meshRenderer = Add<MeshRenderer>();
ModelLoader::Load("path/to/model", meshRenderer);
}
}

How it works: When rendering, it iterates through every Camera in the scene. If the camera is active, it renders to the Camera::target_viewport. Currently, if the Camera::target_viewport is "Main Platform Window", it simply unbinds the frame buffers so that rendering is done to the default frame buffer. This will change in the future. Otherwise, it renders to the frame buffer of specified window. The texture of the specified frame buffer is then passed to imgui for the window that it belongs to by BEImGuiViewport::Render.
