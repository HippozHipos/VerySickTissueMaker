Detailed Description

Represents an object in the scene.

This class can be inherited from by any class that will be in the scene without any behaviour. It inherits from SceneContainer and provides access to the objects stored by ApplicationContainer on top of it.

Each SceneObject also has a Transform object added to it by default.
