Detailed Description

Manages the state of keyboard inputs for the "Main Platform Window" i.e. Window class.

The Keyboard class tracks key press, hold, and release states. It interacts with GLFW through a callback to update key states, and provides methods to query whether keys are pressed or held. It uses a bitset to store the state of all keys.
