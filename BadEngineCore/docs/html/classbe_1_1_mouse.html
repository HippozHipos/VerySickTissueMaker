Detailed Description

Manages the state of mouse inputs for "Main Platform Window" i.e. Window class.

The Mouse class tracks mouse button presses, holds, and movement. It interacts with GLFW to handle mouse position, button states, and scroll wheel movements. The class provides methods to query the mouse state and update internal data accordingly.
